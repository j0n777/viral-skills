# viral-skills

@jonataribas viral content creation skills para Claude Code.

## Instalação

```bash
npx skills add j0n777/viral-skills
```

## Skills

| Skill | Descrição |
|-------|-----------|
| `/viral-hooks` | Gera hooks virais para redes sociais |
| `/viral-carousel` | Cria carrosséis (educativo, storytelling, lista, bíblico) |
| `/viral-reels` | Roteiros para Reels, TikTok, Shorts |
| `/youtube-script` | Roteiros completos de YouTube |
| `/youtube-title-thumbnail` | Títulos + conceitos de thumbnail |
| `/youtube-analyzer` | Análise de canais e vídeos |
| `/viral-trends` | Tendências virais em tempo real |
| `/image-gen` | Geração de imagens com IA |
| `/video-creator` | Criação de vídeo (HeyGen/Descript) |
| `/insta-scraper` | Scraping de conteúdo viral do Instagram |
| `/cross-post` | Cross-posting multi-plataforma |
| `/auto-post` | Automação e agendamento |
| `/content-strategy` | Planejamento e calendário editorial |
| `/niche-bible` | Conteúdo bíblico para redes sociais |

## Uso

Depois de instalar, use os comandos diretamente no Claude Code:

```
/viral-hooks "dicas de produtividade" instagram
/viral-carousel "marketing digital" educativo 7
/viral-reels "receita fácil" 30s
```
