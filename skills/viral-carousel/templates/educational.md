# Template: Carrossel Educativo

## Estrutura (7-10 slides)

1. **Cover:** Pergunta provocativa ou afirmação surpreendente
2. **Problema:** "A maioria das pessoas..." / "O erro mais comum é..."
3. **Contexto:** Por que isso importa / dados que comprovam
4. **Solução 1:** Primeiro passo ou dica prática
5. **Solução 2:** Segundo passo com exemplo
6. **Solução 3:** Terceiro passo com resultado esperado
7. **Resumo:** Recap visual dos 3 pontos
8. **CTA:** "Salve para consultar depois" + perfil

## Regras

- 1 conceito por slide, sem sobrecarregar
- Usar ícones/emojis como bullets visuais
- Progressão lógica clara (problema → solução)
- Último slide SEMPRE tem CTA + @perfil
- Incluir número do slide (ex: "3/7") para criar expectativa
